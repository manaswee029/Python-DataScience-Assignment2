## Assignment 2 - Pandas

### What I learned
- Created a DataFrame from a dictionary
- Used .head() and .describe() for data exploration
- Added a new Bonus column which is 10% of salary
- Filtered employees aged between 25 and 30
- Grouped data by department to find average salaries
- Sorted data and saved to CSV file

### Key Insights
- Marketing pays the most at Rs.62,000 and HR pays the least at Rs.46,000
- Only 2 out of 5 employees fall in the 25-30 age range
- Total bonus paid across all employees is Rs.25,800

### How to Run
Install pandas using: pip install pandas
